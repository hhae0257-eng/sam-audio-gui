================================================================
  SAM-Audio 소리 분리 도구
  (Meta의 SAM-Audio 모델 기반 · 텍스트 프롬프트로 원하는 소리만 분리)
================================================================

■ 이게 뭔가요?
  영상이나 오디오에서 "대사만", "음악만", "특정 효과음만" 뽑아내는 도구입니다.
  MSST(Bandit)처럼 고정된 스템이 아니라, 원하는 소리를 영어 텍스트로
  지정("dog barking", "footsteps" 등)해서 분리할 수 있습니다.

■ 준비물
  - Windows 10/11
  - NVIDIA GPU (권장: RTX 30/40/50 시리즈, VRAM 8GB 이상)
  - 인터넷 (첫 설치 때 파이썬·torch·모델 다운로드)

■ 설치 (처음 한 번)
  1) "install.bat" 더블클릭  → 파이썬·venv·torch(cu128)·SAM-Audio 자동 설치
     * 용량이 크고(수 GB) 시간이 오래 걸립니다. 창을 닫지 마세요.

  2) [중요] SAM-Audio 모델은 Hugging Face 승인이 필요합니다.
     - 접속: https://huggingface.co/facebook/sam-audio-base
       (버튼 눌러 접근 요청 → 보통 자동/빠르게 승인)
     - 로그인(토큰 등록):
         venv\Scripts\hf auth login
       화면 안내대로 HF 토큰을 붙여넣기 하세요.
       (구버전 huggingface-cli login 은 폐기됨 — hf auth login 사용)
       (토큰 발급: https://huggingface.co/settings/tokens)

■ 실행
  - "시작하기.bat" 더블클릭
  - 창이 뜨면 상단 상태등이 초록(GPU 준비됨)이 될 때까지 잠깐 기다립니다.
    (첫 분리 실행 때 모델을 내려받고 메모리에 올리므로 처음은 느립니다.)

■ 사용법
  1) 영상/오디오 파일을 창에 끌어다 놓기(또는 클릭해서 선택)
  2) 위 탭에서 모드 선택:
       - 대사/음성  : 사람 말만 분리
       - 효과음     : 원하는 소리를 영어로 입력 (예: footsteps)
       - 음악/BGM   : 배경음악 분리
       - 고급       : 프롬프트 + 시간구간(스팬) + 재랭킹 직접 제어
  3) "분리 시작" → 결과 2개(target=뽑은 소리 / residual=나머지) 재생·저장
     (영상 입력이면 "리먹스" 체크 시 뽑은 오디오를 영상에 다시 얹어줍니다)

■ 팁
  - 모델 크기: base 권장. 품질 더 필요하면 large(느림), 빠르게는 small.
  - 프롬프트는 영어 소문자 명사/동사구가 잘 됩니다. ("thunder" O / 문장 X)
  - 결과 파일은 results\ 폴더 아래 job_* 에 저장됩니다.

■ 문제 해결
  - 상태등이 빨강/노랑에서 안 바뀌면: install.bat을 다시 실행해 설치를 완료하세요.
  - "GPU 준비 안 됨(CPU)"로 나오면: torch가 cu128로 안 깔린 것입니다.
    RTX 50 시리즈는 반드시 CUDA 12.8 빌드가 필요합니다(install.bat이 처리).
  - 분리가 500 오류면: HF 로그인/모델 승인이 안 된 경우가 많습니다(위 2번).
